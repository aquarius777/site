What is ΣRGO?

A decentralized blockchain platform, based on novel scientific ideas

User-friendly protocol, allowing to run full node on smartphone

Protocol-friendly transactional language, making your money smart

What makes ΣRGO unique among blockchains?

FUNDAMENTAL APPROACH

ΣRGO platform is the combination of modern ideas that complement each other and improving different aspects of cryptocurrency. This ideas were presented on best conferences and have positive reviews from the community.

SCALABILITY

Now there is no silver bullet in blockchain scalability question and ΣRGO is going to support different approaches at the same time. Better on-chain throughput (up to 1000 tx/s) will be available with Bitcoin-ng proposal,while flexible transactional language will allow to build offchain protocols on top of it, increasing throughput by additional factor of 1000. If this still won’t be enough, ΣRGO protocol will allow to lunch more blockchains with sidechains token transfer between them.

LIGHT TRUSTLESS WALLET

ΣRGO will require just few Kb of data to synchronize the network and start to process blocks in trustless manner that allows to build user-friendly mobile wallet with full node security guarantees. This is important for network decentralization, preventing users to use centralized services and exchanges as their wallets.

BLOCKCHAIN AS DATABASE

One of the main problems for blockchain as database usage, is that it is required to download and process all the transactions to find transactions of concrete service. ΣRGO will solve this problem with Aspen protocol, that allows to partition a blockchain such that users can fully validate their services and while keeping track of only the subset of the blockchain that is relevant to their services. ΣRGO ecomony model will make it’s usage as database more flexible, allowing miners to control storage rent in the blockchain.

SMART MONEY

ΣRGO transactional language is the flexible alternative for Bitcoin scripting language, that combines environment conditions and cryptographic Σ-protocols with AND, OR, k-out-of-n connectives. Despite it is not Turing-complete, it allows to build smart contracts, that are effectively not possible in Ethereum-like smart-contract platforms while allowing to perform efficient cost analysis preventing DoS attacks.
